# Project Description
LocAle is a mobile app to discover craft beers and find breweries near you.

# Technos Used
- Frontend : REACT NATIVE
- Backend : NODE.JS / EXPRESS
- Database : MONGO DB
- State container : REDUX
- Component library : NATIVE BASE
- Image API : CLOUDINARY
- Teamwork : GIT / GITHUB
- Framework : SCRUM
